== Description ==

This free and comprehensive refinance calculator can help you calculate monthly payments, amortization, total purchase price, and more.

[https://www.calculator.io/refinance-calculator/](https://www.calculator.io/refinance-calculator/)

__Refinance Calculator Shortcode__

`[ci_refinance_calculator]`

== Installation ==

1. Upload the /ci_refinance_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the [Refinance Calculator](https://www.calculator.io/refinance-calculator/ "Refinance Calculator Homepage") plugin through the "Plugins" menu in WordPress.

== Refinance Calculator Usage ==

* Add the shortcode `[ci_refinance_calculator]` to your page, post or sidebar.
* Add the following code: `<?php display_ci_refinance_calculator(); ?>` to your template where you would like the Refinance Calculator to appear.
